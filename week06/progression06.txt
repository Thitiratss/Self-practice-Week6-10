ใน week06 หนูเอาโจทย์ของอาจารย์ที่ให้มาลองทำโจทย์ก่อนค่ะ โดยเริ่มจากสร้าง array ว่างชื่อ quotes 
จากนั้นหนูมาเขียนฟังก์ชันที่เพิ่ม quotes เข้าไป คือฟังก์ชัน addQuote หนูใช้ if(quote !== undefined || quote !== null) เพื่อเช็กก่อนว่าค่าที่ส่งเข้ามามีจริงไหม เพราะถ้าเราเผลอส่งค่า undefined หรือ null เข้าไป แล้วใช้ .push() เลย จะทำให้ array ของเรามี element ว่างหรือ error ได้ค่ะ 
จากนั้นหนูมาเขียน นฟังก์ชัน deleteQuote หนูใช้ findIndex เพื่อหา index ของ quote ที่มี id ตรงกับที่เราต้องการลบ เพราะเราจะต้องลบ element จาก array การใช้ splice(index, 1) ข้อนี้หนูใช้ AI ช่วยค่ะเพราะว่าแชร์ code กับเพื่อนแล้วไม่เข้าใจ ซึ่ง findIndex จะลบ element ที่เจอจาก index ซึ่งง่ายและเร็วกว่า loop ค่ะ
ต่อมา ฟังก์ชัน updateQuote หนูใช้ find หา object ที่ตรงกับ id ก่อน แล้วค่อยอัปเดต property  content หรือ author หนูใช้ ใช้ if(updatedQuote.content !== undefined) เพื่อเช็กว่าค่าใหม่ถูกส่งมาหรือยัง ไม่เปลี่ยนค่าเดิมถ้าไม่ได้ส่งมา
จากนั้น ฟังก์ชัน getAllQuotes หนู  return quotes ทั้งหมดเพื่อดูค่าใน array ค่ะ ทั้งหมดนี้ทำให้หนูเข้าใจ การจัดการ object ภายใน array มากขึ้นค่ะ
ต่อมา **โจทย์เพิ่มเติม** หนูให้ AI ช่วยสร้างโจทย์ให้ค่ะ โดยอิงจากสไลด์ที่อาจารย์สอนและขอให้ช่วยเพิ่มความยากอีกนิด
หนูได้ลองฝึก higher-order function และ closure โดยเริ่มจากฟังก์ชัน createFilter(minPrice) ซึ่ง return ฟังก์ชันอีกตัวหนึ่งฟังก์ชันที่ return นี้จะกรอง array ของสินค้าให้เหลือเฉพาะสินค้าที่ราคามากกว่า minPrice วิธีนี้เป็นการเขียนแบบ closure ค่ะ function ซ้อน function 
ต่อไปหนูได้เขียนฟังก์ชัน bankAccount(initialBalance) ซึ่ง return object ที่มี method deposit, withdraw, และ getBalance หนูทำให้ฟังก์ชัน เป็น property เพราะตอนแรกไม่รู้ค่ะว่าสามารถทำแบบนี้ได้ และลองทำแบบที่อาจารย์สอนแต่มันไม่สามารถ return ฟังก์ชัน แบบซ้ำ ๆ ได้ มันจะใข้ได้แค่ฟังก์ชันแรกที่เราเขียน เลยถาม AI ค่ะว่าสามารถทำยังไงได้บ้าง การทำให้ฟังก์ชัน เป็น property ทำให้เข้าถึงทั้งสามตัวได้เลยเป็นวิธีที่สะดวกที่สุดและทำให้รู้เป็นครั้งแรกเลยค่ะว่าสามารถเขียนแบบนี้ได้
และข้อฟังก์ชัน getCartSummary(cart) เพื่อรวมยอดรวมของสินค้าในตะกร้า หนูใช้ .reduce() เพราะอยากให้มันวนทุกตัวใน array ทีเดียว แล้วบวกจำนวนสินค้า totalItems กับราคาสินค้าทั้งหมด totalPrice ไปพร้อม ๆ กัน โดยให้ acc เก็บค่าปัจจุบันจากที่เราวนแล้วบวกมาเรื่อย ๆ ในแต่ละรอบของ reduce หนูใช้ item.qty * item.price เพื่้อคำนวณราคาของสินค้าทั้งหมด เช่นถ้ามีหนังสือราคา 120 บาท 2 ชิ้น ราคาของหนังสือตัวนั้นจะเป็น 240 บาทค่ะ
