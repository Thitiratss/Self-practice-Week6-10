ในการทำ self-practice เรื่อง DOM ครั้งนี้ หนูให้ AI ช่วยสร้างโจทย์ให้ประมาณ 5 ข้อใหญ่ แล้วมีข้อย่อยแบ่งไปตาม folderหนูได้เรียนรู้ การดึง <h2>, <p>, <div>, การเช็ค nodeType, nodeName, และ nodeValue เพื่อเข้าใจว่าแต่ละ node ใน DOM tree เป็นประเภทไหน เช่น element, text หรือ comment รวมถึงการสร้าง element ใหม่ เช่น <p>, <ul>, <li>, <span> แล้วใช้ appendChild, replaceChild, insertBefore, และ setAttribute จัดการ DOM บางข้อหนูรู้สึกว่า DOM ค่อนข้างยาก เช่น การสร้าง <ul> และ append <li> "Alice", "Bob", "Charlie" แบบเป็น Array หนูจึงใช้ AI ช่วยอธิบายแนวทางทำให้เข้าใจมากขึ้น หรือบางข้อที่ต้อง insert child ไปบนสุดก็ถาม AI  เพื่อดูตัวอย่างโค้ดเพราะว่ายังไม่รู้ในส่วนนี้ ข้อที่เป็นการ remove ค่าหนูลองใช้ textContent = null ก็ได้ผลค่ะ แต่พึ่งรู้ว่าสามารถใช้ remove ได้ด้วยจาก AI และหนูได้ฝึก loop ผ่าน .childNodes และ .children เพื่อเข้าถึงทั้ง element และ text node ต่าง ๆ และแสดงผลด้วย console.log ซึ่งทำให้เห็นความแตกต่างระหว่าง childNodes และ children ส่วนมากหนูทำ self-practice ตาม scope ที่่อาจารย์สอนเพราะยังเขียน DOM ไม่ค่อยคล่องและมีบางส่วนที่เขียนไม่ได้เลยด้วยค่ะ แต่ก็ทำให้หนูเข้าใจภาพรวมของ DOM มากขึ้นค่ะ การอ้างถึง document เสมอ การใช้ getElementById ทำให้หนูเข้าใจวิธีเข้าถึง element ต่าง ๆ ได้ชัดเจนขึ้น
